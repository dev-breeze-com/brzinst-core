Breeze::OS Installer -- User Account Creation

You must have an account for yourself,
the system administrator [root], and the security manager [secadmin].
The sysadmin or root acccount, as well as,
the secadmin or secadmin account, are automatically created.
The default password for root and secadmin is root.
Change the password for root by clicking here.
Change the password for secadmin by clicking here.
Create user accounts by clicking here.
Only the full name and password are necessary.
The username is derived from the provided full name.

Click on the Next arrow button to continue.
