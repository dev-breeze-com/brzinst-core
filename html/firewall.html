Breeze::OS Installer -- Desktop Network Firewall

You must have a firewall running to prevent network intrusions.
The IP tables filtering layer is used in Linux/BSD systems
to insert firewall rules into the running kernel.
If your internet connection was router; then, you might also
be running a firewall within the router; in that case,
it may not be necessary to enable the desktop firewall.
