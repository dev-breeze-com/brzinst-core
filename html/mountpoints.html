Breeze::OS Installer -- Select Mount Points

Drive: %selected-drive%
Model: %selected-drive-model%
Assuming, you have chosen to update your installation without formatting your target drive.
If you have not; then, you must first partition and format a drive, before continuing.
If yes, you must first fill-in the empty mount point for each partition, below.
The selected mount points must include your root [/] partition.

For LVM partitioned drives you must go through the LVM partitioning steps.
Updating on LVM drives is not enabled in this version.

Click on to Set Mount Points.
