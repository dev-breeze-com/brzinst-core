Breeze::OS Installer -- Xorg X11 Installation

The Xorg X11 toolkit provides the basic functions for your desktop.
Almost every graphical user interface (GUI) relies on it.
The Xorg X11 toolkit tries to automatically detect your devices; but,
configuration files can be used to set a few default settings.
Following is a partial device detection report:

  • Mouse device %detected-mouse-device%
  • VGA controller %detected-vga-controller%
  • Audio device %detected-audio-device%
  • Modem device %detected-modem-device%
  • Wireless apdapter %detected-wireless-adapter%
The screen sync values should be in your monitor manual.
The screen resolution should also be in your monitor manual.
Your monitor name is for informational purposes.
The graphics card entry is used to select an Xorg/X11 driver.
