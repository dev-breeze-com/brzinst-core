Breeze::OS Installer -- Kernel Selection

To the left, is a list of kernels from which to choose.
Some are for Symmetric Multi-processing SMP kernels.
All available kernels are for Intel or Athlon computers.
The default kernel is for modern Intel/AMD computers or compatibles,
which support Physical Address Extension (PAE) such as,
the Intel Pentium Pro/II/III/4/4M/D, Xeon, Core and Atom;
AMD Geode NX, Athlon (K7), Duron, Opteron, Sempron, Turion or Phenom;
Transmeta Efficeon; VIA C7; and some other processors.
You may select a 64-bit kernel, if you are installing on a 64-bit computer.
You may select an SMP kernel for either a 32-bit or 64-bit computer.
Note that some of the wireless kernel drivers only work in 32-bit mode.
You must select a 32-bit kernel, if you have an HCF Conexant modem.

Select a kernel, or click on to Continue.
