Breeze::OS Installer -- Network Package Source

You must select a network entry as package source.
Your selection will be verified as a valid Breeze::OS package source.
If you select an IPv4 address as a package source;
you must then enter the actual address in the IPv4 entry, on the left.
You must select a desktop for installation.
You can select either a desktop or server installation.
A desktop installation consists in installing the list of packages,
according to the selected desktop, onto this computer.
A server installation consists in replicating the server's set
of installed packages onto this computer.
See the file /etc/brzpkg/factory/netinstall.txt.
If master computer is selected as a package source; then,
an install only media must be used; or the set of distro packages
must be replicated on the server computer.
On saving your settings,
the selected server will be contacted, and
the required information retrieved.

Click on to Save your settings.
Click on to Continue.
