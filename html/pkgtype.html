Breeze::OS Installer -- Package Type Selection

You can select the type of package used for installation.

If you are using the Live DVD release; all install packages will be retrieved from the network.
The process could take at least an hour more, than the squashfs installation.
More than an hour, if you have a slow connection.
The default package type is squashfs.





Click on the box icon, to select the install mode of installation.

Click on to Reset to the squashfs type of installation package.

Click on to Continue.
