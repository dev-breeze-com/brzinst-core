Breeze::OS Installer -- Secure Socket Layer (SSL) Encryption Keys

You must have a set of encryption keys to enable SSL connections.

The SSL set of public keys are required by a server to allow
connections from client plugins and applications.

The set of encryption keys are generated, using the information
you must enter in the entry fields on the left.

The default system SSH/SSL keys are used, if a user selected set is not created.

Click on to Create your SSH/SSL keys.
