Breeze::OS Installer -- Boot Loader Settings

You must enable boot configuration by selecting a boot loader.
The boot configuration will allow you to boot your new installation.
It also allows booting a Windows installation, if any present.
The following boot loaders are available:

  • Gummiboot Gummiboot, a UEFI Boot loader.
  • Syslinux the recommended boot loader.
  • Lilo LInux LOader, the classic OS boot loader.
The set of offered boot loaders is based on your partitioning mode,
your root partition filesystem, and your kernel selection.
The Syslinux boot loader only handles
the VFAT, Ext3, Ext4, or Btrfs filesystem, as the boot filesystem.
The Btrfs filesystem is not available in this release.
