Breeze::OS Installer -- Installation Target

You must select the target drive, from the list on the left,
onto which to install software packages.
The target drive must have been partitioned and formatted.
If you formatted the drive; then the proper filesystem mounts will be set.
If you have partitioned several drive(s) without LVM; then,
any of the partitioned drive will do.
If you have partitioned several drive(s) with LVM; then,
select the first LVM partitioned drive.

Select a target drive before Continuing.
