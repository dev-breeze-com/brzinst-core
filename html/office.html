Breeze::OS Installer -- Desktop Installation

A desktop consists of software applications, such as:

a Windowing Environment,
a Desktop Environment,
an Office Suite, and
Networking Tools.

The basic desktop option, is the one you should select;
if you intend to replicate a desktop installation from a master system.
Your selection for network installation must have been
network-client.

The complete desktop installation, not only includes
the Breeze::OS Installer desktop; but, also the Gnome desktop.
