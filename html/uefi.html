Breeze::OS Installer -- UEFI Encryption Keys

You may wish to create your own set of UEFI encryption keys
to enable UEFI booting.

The set of encryption keys are generated, using the information
you must enter in the entry fields on the left.

If you choose not to create your own set of keys then
the default system UEFI keys are used for booting.

Click on to Create your UEFI keys.
Click on to Continue.
