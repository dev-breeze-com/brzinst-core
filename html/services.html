Breeze::OS Installer -- RC Daemon Services

You may enable some of the services, listed on the left.
The essential ones are already enabled.
Important Services
The Printing service provides printing daemons, with cups as the default.
The RNG Daemon service generates random numbers for your computer.
The Network Manager provides a daemon to handle your web connections.
The Mcelog service is used to track computer check errors.

Standard Services
The CPU Frequency service selects the CPU frequency used by your computer.
The Firewall service protects your computer from network intrusions.
The SSH service provides a daemon for secure shell connections.
The Database service provides an SQL database server.

Miscellaneous Services
The NFS service provides a transparent way to share files across a network.
The Samba service allows sharing of files with Microsoft Windows clients.
The NIS service provides a way to share passwords across a network.
The LDAP provides Lightweight Directory Access Protocol services.
The Wireless service enables wireless connections, at boot time.
The ZRAM service provides a compresed ramdrive for memory usage.
The HTTP Web Server service provides the default HTTP server.
The Laptop Utilities service initializes all devices required by laptops.
The Bluetooth service provides a daemon to handle bluetooth connections.
The TFTP Daemon service FTP access for your computer. The Mail Transport Agent provides mail delivery.
