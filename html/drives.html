Breeze::OS Installer -- Drives / Partitioning / Formatting

UEFI stands for Unified Extensible Firmware Interface.
UEFI should only be used, if you have a 64-bit computer
preloaded with Windows, which is newer than 2010.

GPT stands for GUID Partitioning Table.
GPT allows hard drives greater than 2TB to be partitioned.
GPT should be selected for computers newer than 2010.
Windows computers do not work well with GPT.
Only choose GPT partitioning, if your bios supports it.

MBR stands for Master Boot Record.
MBR is the default choice, if you want to run a standalone
system; and, your BIOS does not support UEFI secure booting.

Select a drive before Continuing.
