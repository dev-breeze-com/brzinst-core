Breeze::OS Installer -- Package Source

You must select your source of packages before continuing.

  • Cdrom if your installation media is on a cdrom drive.
  • Flash if your installation media is on a flash drive.
  • Disk if packages are to be retrieved from a disk drive.
  • Network if packages are to be retrieved from the network.
All sources are verified as valid Breeze::OS package sources.

Select a source, or click on to Continue.
