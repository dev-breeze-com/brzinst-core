Breeze::OS Installer -- LVM Logical Volume Formatting

In an LVM environment formatting of logical volumes is similar to formatting hard-drive partitions.
You must, in both instances, select the filesystem with which to format the volume, as well as,
provide a filesystem path or mountpoint. But, you must also format the non-LVM partitions.
If several drives were used; then each boot partition has a different mountpoint.
